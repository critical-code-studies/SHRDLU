About this project

This is a Critical Code Studies reading of SHRDLU, Terry Winograd's natural-language-understanding program for the MIT blocks world (1968–70).

It is one of a series of critical code readings of the founding programs of the long 1960s, alongside the team's readings of Joseph Weizenbaum's ELIZA (1966) and the MIT Spacewar! (1962). The method it follows is set out in Critical Code Studies and in the bibliography.
